FIX FOR VERCEL BUILD ERROR

Problem:
"The pattern api/*.js defined in functions doesn't match any Serverless Functions..."

Fix:
This version removes the custom functions pattern from vercel.json.
Vercel will automatically detect API Routes inside the api/ folder.

GitHub repo root MUST show:
- api/
- public/
- package.json
- vercel.json

Inside api/ MUST show:
- _db.js
- test-db.js
- register-parent.js

Inside public/ MUST show:
- index.html
- app.js
- style.css
- parent-register.html

Vercel settings:
Framework Preset: Other
Build Command: npm run build
Output Directory: public
Install Command: npm install

Environment variable:
MONGODB_URI = your MongoDB connection string

After uploading this to GitHub:
1. Commit changes.
2. Go Vercel.
3. Deployments > Redeploy.
4. Test:
   https://YOUR-SITE.vercel.app/api/test-db
