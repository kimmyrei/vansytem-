VERCEL + GITHUB GUIDE

Use this setup:
GitHub repository -> Vercel project -> MongoDB Atlas

Steps:
1. Upload all files in this folder to GitHub.
2. Go to https://vercel.com
3. Login with GitHub.
4. Click Add New Project.
5. Import your Mutahus Global GitHub repository.
6. Keep default settings.
7. Deploy.
8. Open Vercel Project Settings.
9. Go to Environment Variables.
10. Add:
    Key: MONGODB_URI
    Value: your MongoDB connection string
11. Save.
12. Redeploy.

Test:
https://YOUR-VERCEL-SITE.vercel.app/api/test-db

If success, test:
https://YOUR-VERCEL-SITE.vercel.app/parent-register.html
